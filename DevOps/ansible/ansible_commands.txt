-------  Ansible Playbook run steps   --------

1. ansible --version - To check the ansible state
2. ssh-keygen  - Generating key of the user
3. ssh-copy-id node-username@publicip / ssh-copy-id -i inventory-file-name - Copy the key to the node
4. login to node
    * sudo visudo - Give user Sudo permissions to execute any command without asking for password
    * exit
5. copy / pull / create all the ansible files
6. ansible-playbook -i inventory-file-name playbook-name --syntax-check  -  To check Syntax of the ansible file
7. ansible-playbook -i inventory-file-name playbook-name --check  - To execute playbook with out actually install anything
8. ansible-playbook -i inventory-file-name playbook-name  - To execute playbook
