--------   Git Commands   ----------
git clone "URL" - to download git repository into the remote machine
git init -b main - initializing the main folder to the git repository
git add .  - adding all the folder's and file's to commit changes
git commit -m "Comments for added repository"  - commit changes to the repository with a comment
git push origin main - push code to the main repository
